Getting Started
~~~~~~~~~~~~~~~

Dependencies
^^^^^^^^^^^^

Standard python data analysis and visualization stack: ``pandas``, ``matplotlib``, ``bokeh``

Installation
^^^^^^^^^^^^


Clone the repository and install either in ``conda`` or python virtual environment containing 
the usuail python data analysis stack using ``pip``.

::

    $ git clone https://github.com/eco32i/nlabpy
    $ pip install -e nlabpy


Examples
~~~~~~~~

See example notebooks in ``examples`` directory
